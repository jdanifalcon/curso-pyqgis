Curso Práctico

Introducción a
Python en QGIS

Optimiza tus flujos de trabajo espaciales. Este curso está diseñado para usuarios de QGIS que desean aprender los fundamentos de programación con Python. Pasa de conceptos iniciales a la aplicación de PyQGIS para consultar información, manipular capas y automatizar tareas sencillas.

Objetivo

Introducir el uso de Python aplicado a SIG, desarrollando habilidades fundamentales para interactuar y optimizar flujos de trabajo geoespaciales mediante código.

Dirigido a

Estudiantes, profesionales ambientales, biólogos, analistas SIG, etc. (No se requieren conocimientos previos en programación).

Módulos del Programa
01 Fundamentos de Python desde cero
  • Uso de la consola, ejecución de comandos y creación de variables.
  • Estructuras de datos (texto, números, listas) y condicionales.
02 Introducción a la biblioteca PyQGIS
  • Acceso al proyecto actual (QgsProject) e identificación de capas.
  • Consulta de información general: nombre, elementos y campos.
03 Trabajo con datos geográficos y atributos
  • Lectura de atributos, geometrías y entidades (features).
  • Selección automática y filtros mediante expresiones.
04 Automatización básica en QGIS
  • Creación de scripts para contar elementos y revisar atributos.
  • Proyecto final: Automatización integral de una capa SIG.
