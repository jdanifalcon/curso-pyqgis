2. Variables y Asignación

Una variable es una "caja" con un nombre, donde guardamos un dato. La asignación se hace con el signo de igual (=).

  • Python es de tipado dinámico: no necesitas decirle si es texto o número, él lo adivina.
# Asignación de variables Primitivas
nombre_municipio = "Progreso"   # String (Texto)
poblacion = 66008               # Integer (Entero)
temperatura = 34.5              # Float (Decimal)
es_costero = True               # Boolean (Verdadero / Falso)

3. Operadores Matemáticos

Los símbolos para hacer cálculos matemáticos entre variables numéricas:

  • Suma: +   |   Resta: -
  • Multiplicación: *   |   División: / (Devuelve decimales)
  • Exponente (Potencia): **
  • División entera: // (Quita los decimales del resultado)
# Cálculos utilizando variables
base_lote = 20
altura_lote = 50

area_total = base_lote * altura_lote
print("El área es:", area_total)  # Resultado: 1000

4. Operaciones con Textos (Strings)

¡Los operadores matemáticos también funcionan con otros tipos de datos, como los textos!

  • Concatenación (+): Une dos o más textos.
  • Repetición (*): Multiplica un texto varias veces.
# Uniendo textos (Concatenación)
palabra_1 = "Área de "
palabra_2 = "Conservación"
resultado = palabra_1 + palabra_2
print(resultado)  # Imprime: Área de Conservación

# Repitiendo textos
linea_separadora = "-" * 20
print(linea_separadora)  # Imprime: --------------------

5. Las Listas (Estructuras No Primitivas)

Una Lista te permite guardar múltiples datos dentro de una sola variable. Se crean usando corchetes [ ] y separando los elementos con comas.

  • Pueden contener tipos de datos mezclados.
  • El conteo de posiciones (índices) ¡Siempre empieza desde el CERO!
# Crear una lista de capas
mis_capas = ["Rios", "Vialidades", "Uso_Suelo"]

# Acceder a un elemento (El 0 es "Rios")
print(mis_capas[0]) 

# Agregar un nuevo elemento al final de la lista
mis_capas.append("Curvas_Nivel")

6. Funciones y Paquetes (Librerías)

Funciones: Son acciones prefabricadas que hacen el trabajo duro por ti. Llevan paréntesis ().

Paquetes/Librerías: Son "cajas de herramientas" adicionales que puedes traer a tu proyecto usando el comando import.

# FUNCIONES NATIVAS (Ya vienen listas para usar)
print("Esto es una función")
cantidad = len(mis_capas) # len() cuenta cuántos elementos hay

# IMPORTAR PAQUETES (Traer nuevas herramientas)
import math  # Importa la librería de matemáticas avanzadas
import os    # Importa herramientas para leer carpetas de Windows/Mac

raiz = math.sqrt(64) # Saca la raíz cuadrada usando el paquete math

7. ¿Cómo llegó Python a QGIS? (Breve Historia)

QGIS no siempre fue tan amigable para automatizar. Su evolución fue impulsada por la necesidad de usuarios que querían dejar de hacer tareas repetitivas a mano.

  • El origen (2002): QGIS fue creado en el lenguaje C++. Muy rápido, pero sumamente difícil de aprender para profesionales que no eran ingenieros de software.
  • La Revolución (2007): Con QGIS 0.9, se introdujo oficialmente PyQGIS. El objetivo era democratizar el desarrollo, eligiendo a Python por ser el lenguaje más intuitivo para científicos y analistas.
  • El Boom de los Plugins: Gracias a Python, la comunidad comenzó a crear miles de complementos. Hoy, la gran mayoría de los plugins de QGIS están escritos en Python por usuarios como tú.
  • QGIS 3.x y el futuro (2018-Actualidad): QGIS adoptó Python 3 como su estándar definitivo, permitiendo conectar mapas con inteligencia artificial, Big Data y análisis espacial masivo en cuestión de segundos.
La moraleja: PyQGIS existe porque profesionales de diversas áreas necesitaban una herramienta sencilla para hablar con el motor complejo de QGIS. ¡Ustedes son la nueva generación de esa comunidad!
