La Consola Python

Es la puerta de entrada para interactuar directamente con nuestro mapa usando comandos.

Menú Superior QGIS ➔ Complementos ➔ Consola de Python

Consola (Línea de comandos)

Ideal para pruebas rápidas y ejecutar instrucciones de una sola línea.

Editor de Scripts

Ideal para escribir procesos largos (rutinas) y guardar el código para usarlo mañana.

Nota importante: No es necesario instalar Python adicional en tu computadora para comenzar con PyQGIS. ¡QGIS ya lo trae integrado de fábrica!

Nuestro primer programa

Vamos a pedirle a QGIS que nos devuelva un mensaje de texto. Es el saludo tradicional en programación.

print("Hola PyQGIS")
  • print() es una función. Le dice a la computadora: "Muestra esto en la pantalla".
  • Las comillas (" ") le indican a Python que lo de adentro es un texto, no una instrucción matemática.

Conociendo los objetos principales

En PyQGIS, todo es un "objeto". Estos son los protagonistas de tus mapas:

QgsProject

Representa tu archivo de proyecto (.qgz).
Ej. Tu mapa actual.

QgsVectorLayer

Una capa vectorial (Puntos, líneas o polígonos).
Ej. Ríos.shp.

QgsRasterLayer

Una capa de píxeles/imágenes.
Ej. Un Modelo Digital de Elevación (DEM).

QgsFeature

Una sola entidad (Una fila en tu tabla de atributos).
Ej. Un árbol específico.

QgsGeometry

La forma espacial y coordenadas de una entidad.
Ej. El polígono de un lago.

QgsField

Una columna en tu tabla de atributos.
Ej. Columna "Uso_Suelo".

1. Acceder al Proyecto

El primer paso siempre es decirle a Python: "Quiero trabajar con el mapa que tengo abierto frente a mí".

# Obtenemos la instancia del proyecto actual
project = QgsProject.instance()

# Imprimimos la ruta donde está guardado el archivo
print(project.fileName())
El comando QgsProject.instance() es la llave maestra. Guarda todo el proyecto en una "caja" (variable) llamada project para que podamos consultarlo cuando queramos.

2. Obtener la capa activa

Si tienes 20 capas en tu mapa, la forma más fácil de trabajar con una es seleccionarla con el ratón y decirle a Python que use "la que está activa".

# 'iface' es la interfaz de QGIS. Le pedimos la capa activa.
layer = iface.activeLayer()

¿Qué devuelve esto?

  • Devuelve un objeto de tipo QgsVectorLayer o QgsRasterLayer.
  • Si no tienes ninguna capa seleccionada en tu panel, devolverá None (Nada/Error).

3. Explorando una capa

Una vez que tenemos la capa en nuestra variable layer, podemos hacerle preguntas.

# Imprimir el nombre de la capa
print(layer.name())

# Preguntar cuántas entidades (filas) tiene
total = layer.featureCount()
print("Total de elementos:", total)

Ejemplo práctico: Imagina que necesitas contar cuántos árboles hay en 50 capas distintas. En lugar de abrir 50 tablas de atributos, Python lo cuenta por ti en microsegundos.

4. Recorriendo entidades (El bucle 'for')

La verdadera magia de la automatización está en hacer que Python revise fila por fila tu tabla de atributos, sin que tú tengas que dar clic.

for feature in layer.getFeatures():
    # Aquí adentro, Python repetirá el código por cada elemento
    print("Revisando un elemento...")

5. Accediendo a atributos

Cuando Python está visitando un elemento específico (feature), podemos pedirle que lea cualquier columna de su tabla.

for feature in layer.getFeatures():
    # Acceder a un campo usando el nombre de la columna entre [" "]
    nombre = feature["Especie"]
    
    # Obtener el ID interno de la fila
    id_fila = feature.id()
    
    print(id_fila, nombre)
Python lee la tabla de atributos como si fuera un diccionario. Solo necesitas saber el nombre exacto de la columna para extraer la información.

6. Trabajando con geometrías

Además de la tabla, podemos medir y analizar la forma dibujada en el mapa.

geom = feature.geometry()  # Extraemos la forma geométrica

area_total = geom.area()     # Calcula el área (Si es polígono)
longitud = geom.length()   # Calcula la distancia (Si es línea)
punto = geom.asPoint()     # Extrae X, Y (Si es punto)

El Sistema de Referencia (CRS / EPSG)

Antes de automatizar cálculos, debemos recordar que QGIS proyecta la Tierra sobre un plano. Dependiendo de cómo lo haga, usamos Metros o Grados.

¿Qué es el EPSG?

Es un código numérico global que identifica un sistema de coordenadas.

Por ejemplo:
EPSG:4326 = Coordenadas geográficas (Grados).
EPSG:32616 = UTM Zona 16N (Metros).

Transformación (Transform)

En PyQGIS, podemos indicarle a Python que transforme las coordenadas de un punto de un EPSG a otro al vuelo, sin modificar nuestro archivo original.
